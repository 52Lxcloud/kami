# Kami

Status: Alpha

v3 版本主要重构架构设计，风格不会有大的变化。配置全部动态化，让配置更加简易。

**你的下一个博客何必是博客**

Kami 是为 Mix Space Server 打造的前端。使用 NextJS 开发。

依赖：Mix Space Server >= v3.12.2
